# Sales Prediction using Machine Learning

## Overview

Sales forecasting plays an important role in business planning and marketing strategy. This project predicts product sales based on advertising expenditure across different media channels such as TV, Radio, and Newspaper.

A Machine Learning model was developed to analyze advertising investments and estimate future sales performance.

## Objectives

* Analyze advertising data.
* Understand the impact of different advertising channels.
* Build a machine learning model for sales prediction.
* Evaluate model performance using standard regression metrics.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Joblib

## Dataset Features

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales (Target Variable)

## Project Workflow

1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Correlation Analysis
5. Feature Selection
6. Model Training
7. Model Evaluation
8. Sales Prediction
9. Model Saving

## Model Used

Linear Regression

## Results

* Mean Absolute Error (MAE): 1.461
* Mean Squared Error (MSE): 3.174
* R² Score: 0.8994

The model successfully predicted sales with high accuracy and demonstrated a strong relationship between advertising expenditure and product sales.
